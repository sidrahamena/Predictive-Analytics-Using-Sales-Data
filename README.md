# Predictive Analytics Using Sales Data  
An interactive analytics project focused on forecasting future retail sales trends using historical data, machine learning, and Power BI visualizations to support data-driven decision-making.

## 📊 Quick Peek  
The project combines trend analysis and predictive modeling to uncover future business patterns:

- **Historical Trend Analysis:** Visualizes monthly sales performance and seasonal fluctuations over time.
- **Sales Forecasting:** Uses Linear Regression and Power BI forecasting tools to predict future sales trends.
- **Performance Insights:** Includes KPI cards for total sales, average monthly sales, and predicted future growth.
- **Interactive Dashboard:** Features dynamic filters, category-wise analysis, regional performance tracking, and forecast visualizations.

## 🛠️ Built With  
- **Python** (Pandas, Scikit-learn, Matplotlib)  
- **Power BI**  
- **Superstore Sales dataset from Kaggle**
